Configuração da Chave API

Siga os passos abaixo para ativar as funcionalidades interativas do curso, como a IA Kairos.

Como Gerar sua Chave Gratuita:

  1. Abra o Google AI Studio em uma nova aba.
  2. No menu à esquerda, clique em "Get API key".
  3. Clique no botão azul "Create API key in new project".
  4. Copie a chave secreta que será gerada.
