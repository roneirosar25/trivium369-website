Guia Visual para Gerar sua Chave API

Passo a Passo Detalhado

Siga este guia para gerar sua chave API gratuita do Google AI Studio. É um processo rápido que lhe dará acesso total às funcionalidades interativas do curso.

Passo 1: Acesse o Google AI Studio

O primeiro passo é visitar o site oficial. Abra o Google AI Studio em uma nova aba e, se necessário, faça login com sua conta Google.

Passo 2: Encontre a Opção "Get API key"

No menu à esquerda da tela, localize e clique na opção "Get API key" (Obter chave de API).

Passo 3: Crie a Chave em um Novo Projeto

Uma janela chamada "API keys" irá aparecer. Clique no botão azul "Create API key in new project".

Passo 4: Copie sua Chave Secreta

O sistema irá gerar instantaneamente sua chave API. Clique no ícone de cópia ao lado da chave para garantir que você a copiou corretamente.

Passo 5: Volte e Insira a Chave no Curso

Com a chave copiada, volte para a nossa página de configuração, cole a chave no campo indicado e clique em "Salvar Chave".
